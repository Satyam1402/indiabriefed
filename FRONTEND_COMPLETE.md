---

## 🚀 HOW TO START

### Step 1: Make sure Backend is Running

```bash
# In one terminal
cd D:\indiabriefed\backend
php artisan serve
```

Keep this running at http://localhost:8000

### Step 2: Start Frontend

```bash
# In another terminal
cd D:\indiabriefed\frontend
npm run dev
```

**Frontend will start at:** http://localhost:3000
